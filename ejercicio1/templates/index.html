Actividad 4 - Ejercicio 1 - Inicio

Agregar al final de la url de esta página lo siguiente para obtener diferentes resultados:

  • '/dado' => Genera un número aleatorio entre 1 y 6
  • '/dado/n' => Genera n números aleatorios entre 1 y 6 (n debe ser un numero entero positivo y menor a 10 ingresado por el usuario)
  • '/fecha' => Genera una fecha aleatoria entre el 01/01/1980 y 31/12/2100
  • '/fecha/y' => Genera una fecha aleatoria entre el 01/01/y y 31/12/y (y debe ser un número entero positivo ingresado por el usuario)
  • '/fecha/y/m' => Genera una fecha aleatoria entre el 01/m/y y 31/m/y (y y m deben ser números enteros positivos ingresados por el usuario)
  • '/color' => Genera un color aleatorio entre las siguientes opciones: rojo, azul, verde, amarillo, negro, violeta, naranja o marrón
